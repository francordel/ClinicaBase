# Clínica Base

Landing page moderna y profesional para Clínica Base, un centro de fisioterapia deportiva y rehabilitación en Murcia.

## Descripción

Este proyecto es el sitio web oficial de Clínica Base. Diseñado para ofrecer una experiencia de usuario fluida y atractiva, permite a los pacientes conocer los servicios, el equipo y reservar citas fácilmente a través de WhatsApp.

## Características

-   **Diseño Responsivo**: Adaptado a móviles, tablets y escritorio.
-   **Secciones**:
    -   **Hero**: Presentación impactante con llamada a la acción.
    -   **Servicios**: Tarjetas con iconos personalizados para Fisioterapia General, Deportiva y Rehabilitación.
    -   **Equipo**: Perfiles de los fisioterapeutas (Guille y Fran) con integración directa a WhatsApp.
    -   **Testimonios**: Opiniones de pacientes satisfechos.
    -   **Ubicación**: Información de contacto y dirección.
-   **Estilo Visual**: Paleta de colores corporativa (Azul y Rosa suave), tipografía 'Outfit' y animaciones suaves.

## Tecnologías Utilizadas

-   HTML5
-   CSS3 (Variables CSS, Flexbox, Grid)
-   JavaScript (Vanilla)

## Cómo Ejecutar

1.  Clona el repositorio.
2.  Abre el archivo `index.html` en tu navegador web.
3.  Opcionalmente, puedes usar un servidor local:
    ```bash
    python3 -m http.server 8080
    ```
    Y acceder a `http://localhost:8080`.

## Estructura del Proyecto

-   `index.html`: Estructura principal de la página.
-   `css/style.css`: Estilos y diseño visual.
-   `js/main.js`: Lógica para el menú móvil y animaciones.
-   `assets/images/`: Imágenes y recursos gráficos.
